# MPC-PINN-2k

## Description
To facilitate reproducibility and future research, we release the **2k-parameter distilled PINN model** and dependent **ResNet Functionals**. This repository provides resources for researchers working on Physics-Informed Neural Networks (PINNs) and related computational frameworks.
